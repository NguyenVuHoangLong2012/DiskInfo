# DiskInfo
Name: DiskInfo  
Version: 1.0  
Platform: Windows  
A small utility to check basic information about your drive right in the command line interface.
## Usage:
After adding DiskInfo.EXE to the path environment variable or System32 directory, you can use DiskInfo.EXE to view all current information about your drive, the /n prefix flag to only display a list of drives and their labels, and the /m prefix flag to only display drives available on your computer without labels.
